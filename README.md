# 📝 Todo List

코드잇 스프린트 과제로 진행한 **Todo List 웹 애플리케이션**입니다.  
Next.js(App Router)와 TypeScript를 기반으로 할 일의 **생성, 수정, 삭제, 상태 관리(진행/완료)** 및  
**이미지·메모 첨부 기능**을 구현했습니다.

반응형 UI를 적용하여 **모바일 / 태블릿 / 데스크탑 환경**에서 모두 사용할 수 있습니다.

## 📦 주요 기능

### ✅ 할 일 관리

- 할 일 생성 / 수정 / 삭제
- 진행 중(TODO) ↔ 완료(DONE) 상태 전환
- 상태 변경 시 실시간 UI 반영

### 🖼 이미지 첨부

- 할 일당 이미지 **1개 업로드 가능**
- 파일 제한:
  - 파일명: 영문만 허용
  - 파일 크기: **5MB 이하**
- 이미지 업로드 후 서버에 저장되어 새로고침 시에도 유지

### 📝 메모 기능

- 각 할 일에 메모 추가 및 수정 가능
- 상세 페이지에서 바로 편집 가능

### 📱 반응형 UI

- 모바일 / 태블릿 / 데스크탑 대응
- 화면 크기에 따라 버튼, 레이아웃 자동 조정

---

## 📄 페이지 구성

### 1️⃣ 할 일 목록 페이지 (`/`)

#### 📌 목록 조회

- 진행 중(TODO) / 완료(DONE) 목록을 분리하여 표시
- 로고 클릭 시 언제든 목록 페이지로 이동

#### ➕ 할 일 추가

- 상단 입력창에 할 일 입력
- **Enter 키 또는 추가 버튼 클릭** 시 할 일 생성
- 생성된 할 일은 목록에 즉시 반영

#### ✅ 상태 변경

- 할 일 왼쪽 체크 버튼 클릭 시
  - TODO → DONE
  - DONE → TODO

---

### 2️⃣ 할 일 상세 페이지 (`/items/{itemId}`)

#### ✏️ 할 일 수정

- 할 일 항목 클릭 시 상세 페이지로 이동
- 다음 항목 수정 가능:
  - 할 일 이름
  - 진행/완료 상태
  - 메모
  - 이미지 첨부
- `수정 완료` 버튼 클릭 시:
  - 서버에 저장
  - 할 일 목록 페이지(`/`)로 이동

#### 🗑 할 일 삭제

- `삭제하기` 버튼 클릭 시 할 일 삭제
- 삭제 후 할 일 목록 페이지(`/`)로 이동

## 🌐 API 연동

본 프로젝트는 코드잇에서 제공한 Todo API를 사용합니다.

### 사용 API

- `GET /items` : 할 일 목록 조회
- `POST /items` : 할 일 생성
- `PATCH /items/{id}` : 할 일 수정
- `DELETE /items/{id}` : 할 일 삭제
- `POST /images/upload` : 이미지 업로드
